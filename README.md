# AI Drawing Dashboard (WebUI) 🎨

一个基于 Web 的轻量级 AI 绘图管理工具，支持高级参数调整、历史记录管理和实时速度监控。

## ✨ 特性 (Features)

- **多源支持**: 支持 Liaobots 和 SillyDream (Banana) API。
- **实时监控**: 内置下载速度曲线图与任务状态看板。
- **历史管理**: 本地存储生成记录，支持精选、搜索和隐藏。
- **交互增强**: 支持图片拖拽排序、参考图上传。
- **隐私安全**: API Key 仅存储在本地浏览器，不会上传服务器。

## 🚀 如何使用 (How to use)

1. 下载打开或直接访问https://wilsonandwho.github.io/ai-drawing-dashboard/。
2. 在侧边栏填入对应的 API Key（API Key请自行前往服务商处获取）。
3. 输入提示词，点击 "开始任务"。

## 🛠️ 安装 (Local Run)

只需下载 `index.html` 并用 Chrome/Edge 浏览器打开即可，无需安装 Node.js 或 Python 环境。
